## Area - Charities

### Issue 1: Description starts with name / year
The charity description in some cases starts with the name, and talks about the founding year. When viewed on map, list, and other cards, this makes it redundant, since the user is reading the charity name already in the title and wastes space without explaining what the charity actually does. The size of the charity is also relevant, but should not be in the first sentence since the first sentence should focus on what it does. 


Examples of bad beginning descriptions:
- Colorado Coalition for the Homeless: Colorado Coalition for the Homeless has worked to end homelessness in Colorado since 1985. One of the largest comprehensive homeless services organizations in the state, CCH serves through an integrated model of housing, healthcare, and supportive services.
- Denver Center for the Performing Arts: Denver Center for the Performing Arts (DCPA) is Colorado's largest nonprofit performing arts organization, dedicated to creating live theatrical experiences for audiences across the region. Founded in 1972, DCPA presents Broadway touring productions, produces its own award-winning theatrical season, and runs extensive education programs reaching students statewide.
- Urban Peak ignites the potential in youth experiencing homelessness to exit the cycle and create self-determined lives. Founded in 1988, the organization serves young people ages 12–24 across the Denver metro area, providing everything from street outreach to emergency shelter, transitional housing, and long-term case management.
	- Notes: "Ignites" Feels too branded in their own terms. Also, this is unique in that it fights homelessness for youth specifically, which is a differentiator and doesn't feel clear enough.

Examples of good:
- Food Bank of the Rockies: Sources and distributes food to individuals and families facing hunger across Colorado and Wyoming through a network of Hunger Relief Partners — local pantries, shelters, schools, and community organizations that serve their own neighborhoods.
- Denver Kids Inc: Provides long-term, relationship-based mentoring and advising to students in Denver Public Schools who face barriers to academic success. Denver Kids advisors are embedded directly in schools across the city, supporting students from middle school through high school graduation and into postsecondary life.

#### Fix: Warren Village "good example" starts with org name

#### Add to "Avoid" examples

```
❌ "Urban Peak ignites the potential..." (branded/marketing voice — write factually, third person)
❌ "One of the nation's largest..." (leads with superlative size)
❌ "Boys & Girls Clubs has served youth... since 1961" (leads with org name + founding year)
```

#### Add: Location conventions subsection (after Stage 5 — Locations)

```
**Location label conventions:**
- Use "[Org Name] HQ" for the primary administrative location — not "Administrative Headquarters," "Main Office," or the bare org name.
- Labels should be specific and descriptive: "Lawrence Street Shelter" not "Main Shelter."

**Location description conventions:**
- Do not open with the location name as the subject. If the label is "Edison School Community Garden," don't start with "The Edison School Community Garden is..."
- Do not reference the location by its proper name within the description. Use "members of the community garden" not "members of [Garden Name]."
- Using the org name after the first sentence is fine (e.g. "Denver Urban Gardens grew from...").
- No promotional adjectives: "beautiful," "stunning," "small but mighty," "welcoming," "unique."
- Descriptions are third-person, factual, neutral — not the org's own voice.
- Do not include origin stories or history unless directly relevant to current operations.
- Remove exact counts and stats (plot counts, percentages); those belong in the impact field.
```

#### Update: Description rule — org name usage

Current rule says "do not open with" the org name. Clarify that using the org name after the first sentence is fine and preferred over "the organization."

Add to the description guidance:
```
- The first sentence should not start with the org name or founding year. After the first sentence, use the org name naturally — don't substitute "the organization."
```
